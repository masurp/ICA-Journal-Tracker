# ICA Journal Tracker

A lightweight, automatically updated dashboard tracking the most cited, trending, and latest papers from 16 communication science journals across four publishers.

**Live site:** [masurp.github.io/ICA-Journal-Tracker](https://masurp.github.io/ICA-Journal-Tracker/)

## Features

- **Most Cited / Trending toggle** — switch between all-time citation rankings and papers gaining traction in the last 2 years
- **Latest** — most recently published articles per journal
- **Cross-journal search** — find papers by keyword or author name across all 16 journals at once
- **Keyword Trends** — bar chart of most frequent title words or AI-generated topic keywords across all journals
- **Author links** — click any author name to open their Google Scholar profile
- **Topic chips** — concise topic keywords generated by Claude AI from each paper's title and abstract

---

## How it works

A [GitHub Actions](.github/workflows/update_data.yml) workflow runs every **Monday at 06:00 UTC** and:

1. Queries the **[Crossref API](https://www.crossref.org/)** for the most cited, trending, and most recently published papers per journal (filtered by ISSN), including abstracts where available
2. Generates 3–4 concise topic keywords per paper via **Claude Haiku** (Anthropic API), using title + abstract; results are cached in the JSON files so already-known papers are not re-enriched
3. Writes static JSON files to `data/` and commits them back to `main`
4. GitHub Pages serves the updated site automatically

---

## Data sources & notes

- **Most Cited** — ranked by total citation count from Crossref (`is-referenced-by-count`)
- **Trending** — papers published in the last 2 years, ranked by citation count
- **Latest** — sorted by publication date (Crossref)
- **Topic keywords** — generated by Claude Haiku from each paper's title and abstract (Crossref); cached per DOI so only new papers incur an API call on each weekly run

> **Note on citation counts:** Crossref only counts citations from papers with registered reference lists, which systematically undercounts relative to Semantic Scholar or Google Scholar. The rankings are consistent within this tracker but will differ from other sources.
